# 🎰 Gerador de Apostas: Mega da Virada

Este é um script Python simples para gerar combinações de números para sorteios da Mega-Sena. Ele garante que os números de cada jogo sejam únicos e organizados em ordem crescente.

## 📝 Funcionalidades
* **Personalização:** Permite escolher quantas dezenas quer jogar (ex: 6) e quantos jogos deseja gerar.
* **Lógica Anti-Repetição:** O sistema valida se o número já foi sorteado para a mesma cartela antes de adicioná-lo.
* **Ordenação Automática:** Os números são entregues em ordem crescente, facilitando o preenchimento do volante.



## 🛠️ Tecnologias
* **Python 3**
* **Biblioteca `random`**: Utilizada para a geração de números aleatórios entre 1 e 60.

## 🚀 Como Executar
Basta rodar o script no seu ambiente Python:
```bash
